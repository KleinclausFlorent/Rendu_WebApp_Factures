Voici le rendu de ma webApp

La page de connexion à la webApp est  Connexion.php

Vous pouvez vous créer votre propre identifiant à partir de la page New_ID.php

Le menu vous permet d'accèder aux différentes fonctions de la webApp.

Les formulaires de recherche fonctionnent si on rempli seulement un champ avant d'appuyer sur le bouton.

Il n'y a aucune feuille de style dans mon projet parce que j'ai concentré mon temps sur les requêtes et le php.
